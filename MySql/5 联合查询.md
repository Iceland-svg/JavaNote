## 5.1 一个普通的联合查询是啥样的
### 比较好的步骤：
（1）求笛卡尔积
（2）引入连接条件
（3）根据实际需求添加条件 order by ,limit offset等
（4）针对列进行筛选
### 经典的案例：

```sql
select student.name,course.name from student,course
where student.student_id = course.student_id;
```

```sql
select student.name,course.name from student
join course on student.student_id = course.student_id;
```
